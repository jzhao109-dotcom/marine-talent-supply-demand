# 海洋人才供需匹配分析

通过 DeepSeek 大模型从供需两端文本中抽取能力，经开放编码归一化为中观规范技能，最终在规范技能层面进行百分比匹配，识别海洋人才供给与需求的能力结构错配。

## 研究概览

- **供给端**：141 份高校培养方案、课程体系、专业建设文本 → 提取 2711 条能力条目
- **需求端**：1580 条海洋相关岗位描述 → 提取 2514 条能力条目
- **编码**：4328 个原始能力表达 → 开放编码归并为 40 个规范技能
- **匹配**：分别计算供需内部百分比，以差值百分点衡量错配方向与程度

## 核心发现

**需求高于供给（产业急需）**：船舶建造制造与质量、海上风电工程、海洋工程项目建设管理、港航水运工程、船舶电气自动化

**供给高于需求（培养过剩）**：科研实验创新实践、数字化工具与软件应用、电子通信与自动控制、团队沟通国际化

核心结论：培养端偏"基础与综合培养逻辑"，需求端偏"岗位与工程应用逻辑"，两端存在能力结构错配。
